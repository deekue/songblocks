Song Blocks

A no-tech, toddler-friendly controller for Sonos using NFC tags

Download this project as a .zip file Download this project as a tar.gz file

Welcome to Song Blocks

I built Song Blocks so my toddler could play music on the living room Sonos by himself.

Wood blocks containing NFC tags magnetically attach to a drawer face, behind which sits a Raspberry Pi and an NFC controller. Each block's NFC tag is mapped to a song, which plays when the block attaches and stops when it is detached. The title of each played song is tweeted at @songblocks.

The primary design goal was to make the technology disappear completely: the final experience is organic, tactile and aural - and quite magical. The blocks align and pull themselves to the drawer face, snapping on with a satisfying clack!. Music fills the room until the block is prised away. The @songblocks Twitter feed silently records a history my son's listening habits. It will be fun to graph it in a few months to see how his preferences have changed over time.

Technologies Used

  • Raspberry Pi
  • NFC controller board
  • NFC tags
  • Rare-earth magnets
  • A simple Python script (~100 lines) I wrote, hosted here
  • Open-source Python libs: nfcpy for the NFC interface, TweetPony for the Twitter interface

Making the blocks

I used 3/8" thick walnut boards and cut them down to pieces that were about 1-1/2"x3-1/4".
Cavities were routed out for the NFC tag and magnets The halves were then glued and clamped Each side was cleaned up with a band saw, taking just a thin slice off each side to even things up The edges were rounded off with a router The sides and edges were sanded (the seam between the halves is barely visible) The numbers were burned in I've opted to leave the wood natural - no oil, sealant, etc.

The drawer

Cavities were routed out from the inside of the drawer face to place the magnets The NFC controller board is stuck on to the inside of the drawer face with electrical tape The Raspberry Pi just sits in the drawer, with its power cord running out the back
