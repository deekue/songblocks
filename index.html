Song Blocks

A no-tech, toddler-friendly controller for Sonos using NFC tags

Welcome to Song Blocks

I built Song Blocks so my toddler could play music on the living room Sonos by himself. Wood blocks containing NFC tags and magnets mount to a drawer face on a sideboard behind which are matching magnets, an NFC controller and a Raspberry Pi embedded computer. Each block's NFC tag is mapped to a song which plays when the block attaches, and the song title is tweeted at @songblocks. A primary design goal was to make the technology disappear completely: the experience is purely organic and aural - and quite magical.

Technologies Used

  • Raspberry Pi Model B+ with a WiFi adapter
  • This NFC controller
  • NFC tags
  • Rare-earth magnets
  • A simple Python script (~100 lines) I wrote, hosted here
  • Open-source Python libs: nfcpy for the NFC interface, TweetPony for the Twitter interface
